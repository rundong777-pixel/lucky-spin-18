# Lucky Spin 18 - 游戏设计文档

## 游戏概述
- **名称**：Lucky Spin 18
- **类型**：休闲益智抽奖游戏
- **平台**：网页版（后续App）
- **目标用户**：全年龄段，全球用户

## 核心玩法

### 游戏界面
- 18个格子围成一圈
- 中间显示骰子
- 底部有"掷骰子"按钮
- 显示当前积分

### 游戏规则
1. 初始赠送10积分
2. 点击按钮掷骰子（消耗1积分）
3. 骰子显示1-6，棋子走相应步数
4. 停在格子上获得该格子积分
5. 停在第8格（抽奖格）进入抽奖

### 绕圈机制
- 走完18格算一圈
- 每圈结束后格子积分随机打乱
- 抽奖格位置也随机变化

## 抽奖系统

### 奖品设置
| 等级 | 奖品 | 概率 | 价值 |
|------|------|------|------|
| 特等奖 | iPhone 16 Pro | 0.01% | ¥8999 |
| 一等奖 | AirPods Pro 2 | 0.5% | ¥1999 |
| 二等奖 | 小米充电宝 | 5% | ¥99 |
| 三等奖 | 手机支架 | 20% | ¥29 |
| 参与奖 | 10积分 | 74.49% | - |

### 中奖流程
1. 用户抽中实物奖品
2. 填写收货地址
3. 后台显示待发货
4. 工作人员发货并标记
5. 用户确认收货

## 经济系统

### 积分获取
| 方式 | 获得积分 |
|------|----------|
| 初始赠送 | 10积分 |
| 看广告 | +5积分 |
| 格子奖励 | 0-5积分 |
| 内购 | 1元=10积分，6元=100积分 |

### 积分消耗
- 掷骰子：1积分/次

## 登录系统

### 支持方式
- Apple Sign-In（iOS）
- Google Sign-In（安卓/网页）
- Facebook Login
- 手机号+验证码
- 微信登录（中国）

## 后台管理

### 功能模块
1. **用户管理**：查看所有用户信息
2. **中奖管理**：查看中奖列表，标记发货状态
3. **数据统计**：DAU、收入、转化率
4. **配置管理**：调整概率、格子设置

## 技术架构

### 前端
- React + TypeScript
- Canvas/WebGL 动画

### 后端
- Node.js + Express
- MongoDB 数据库

### 部署
- Vercel（前端）
- MongoDB Atlas（免费数据库）
- 或 Railway/Render（免费后端）
